# Changelog

All notable changes to DependsiT Markdown Studio are documented here.

## [1.0.0] — 2026-07-20

### Added
- Initial release of DependsiT Markdown Studio.
- Privacy-first client-side design: runs entirely in the browser with no servers or tracking.
- Drag-and-drop import support for PDF, DOCX, PPTX, XLSX, CSV, HTML, JSON, XML, EPUB, and RTF files (powered by WebAssembly/Pyodide).
- Live split-pane Markdown editor with formatting toolbar and preview.
- Rich renders: GFM tables, KaTeX math equations, and Mermaid diagrams with PNG/SVG export.
- Tabbed workspace with reordering, duplication, and IndexedDB autosave persistence.
- Document statistics (readability scoring, word counts) and writing goal tracker.
- Built-in themes: Light, Sepia, and Dark modes.
- Distraction-free Focus (Zen) writing mode.
- Installable Progressive Web App (PWA) with full offline support.
- Print-optimized CSS formatting.
